# AESIA Better Guides

A lightweight **Next.js + Fumadocs** MVP that reorganises the AESIA supervision guide corpus into a browsable documentation experience.

## What it includes

- introduction and framing
- official sources / references
- deep summary of the corpus
- one page per transcribed AESIA guide
- checklist and examples section

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Source material

This project was assembled from markdown transcriptions of the official AESIA guide set. The official PDFs remain the legal source of truth.
